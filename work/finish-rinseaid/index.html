Finish Rinse Aid

Role: Generalist
Studio: The Mill
Year: 2018

This project I was tasked to create a digital version of the famous water printer in Osaka that could be used in the advert. Originally just setting up the FX in Houdini I ended up taking the CG from start to finish. It was a completely procedural setup that was faithful to the real life version.

Theme and color theme licensed under MIT.
Built with Zola using anemone theme, suCSS framework & veqev.
