I've been meaning to write something here but I got busy. Busy in the sense of making things. The thing is though I'm not quite sure what to make of it...

I've been using Claude Code pretty consistently since I signed up to it. Here's some of what I've been up to:

- Built more web apps and dahsboards than I ever have before
- Made a personal workout logger for a friend tailored to his unique way of tracking
- Ive made a custom room configurator for a company as a proof of concept with 360 panoramas, a custom backend CMS and deployable on a VPS via docker
- I've designed a whole new Claude CLI tool for use in environments where you might not be allowed to install Claude Code natively
- I've cooked up countless little helper scripts at work for various things

So with all of this, what have *I* actually written in code terms? Pretty much nothing, I've only written plain English to describe what I want.

I'm not sure how I feel about this, and in general I'm wondering if I even remember how to code at all 😆 In spite of all this, it's still amazing how fast you can "ship" via AI and it's hard to imagine being able to do all this without it in any sort of similar timeframe.

One thing I've found immensely useful is building out a plan quickly when you know what you want and are just looking for something to essentially help you organise your thoughts into an action plan. It's fast at spotting potential mistakes, you can rely on your own domain knowledge to know if it's going to be an issue or not.

I'm still finding ways to use it better, keep it consistent between sessions and slowly there's a workflow developing that seems to produce between sessions something that resembles coherence. In the meantime... I think it's time I pick up a "hand coded" project to keep me sharp!
